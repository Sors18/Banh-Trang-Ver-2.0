## 📝 プロジェクト概要 
このプロジェクトは、ベトナムの人気スナックである「ライスペーパー（Banh Trang）」を販売するためのEコマースプラットフォームです。ユーザーフレンドリーなインターフェースと、効率的な管理システムを目標に開発しました。

### 🚀 主な機能 
- **ユーザー向け**: 商品一覧の表示、詳細確認、ショッピングカート機能、注文手続き。
- **管理者向け**: 在庫管理、注文ステータスの更新、売上統計の確認。

### 🛠️ 使用技術 
- **Backend**: ASP.NET Core (C#)
- **Frontend**: HTML5, CSS3, JavaScript (Bootstrap 5)
- **Database**: SQL Server
- **Tools**: Visual Studio, Git
